국어

－문학（시） :

  1. 제목을 읽는다.
  2. 시어의 의미를 긍정／부정으로 나눠본다．
  3. 시의 상황，배경，정서，태도 등을 파악한다．
  4. 보기나 선지를 바탕으로 이해하는 연습도 한다．


－문학（소설） :

  1. 인물마다 넘버링을 하고 관계를 파악하며 읽는다．
  2. 인물, 사건, 배경을 잘 파악한다.
  3. 긍정적 / 부정적 의미의 단어들을 표시한다.


－문학（고전） :

  1. 고전 어휘들을 외운다.
  2. 최대한 많은 작품을 접해본다.
  3. ★항상 문학은 작품을 풀고 나서 분석하고 복습한다★


－비문학 :

  1. 글을 구조화하거나 이미지화하면서 완벽히 독해한다.
  2. 선지들은 지문에서 명확한 근거를 찾는다.
  3. 선지를 판단할 때, 앞뒤로 나누어 판단함으로써 매력적인 오답을 거른다.





























































수학

:

  1. 개념을 탄탄히 다진다.
  2. 처음에는 문제를 풀 때 5분정도만 고민해보고 몰랐던 개념을 정리한다.
  3. 어려운 문제를 풀 때는, 문제의 조건을 분석하면서 어떤 개념을 적용할지 생각한다.
  4. 어느정도 경험이 쌓인다면, 답지를 보지 않고 어려운 문제를 끝까지 풀어본다.





























































영어

:

  1. 영어의 기본은 단어! 단어를 열심히 외운다. (똑같은 단어책 3번정도 반복하기)
  2. 기본적인 문법을 다진다.
  3. 문법을 바탕으로 구문들을 분석하고 내용을 이해해가며 기출을 푼다.
  4. 틈틈히 듣기도 공부하고, 시간을 재고 모의고사도 풀어본다.





























































통합과학

:

  1. 개념을 정말 정말 잘 정리하고 외운다.
  2. 문제를 엄청나게 풀고, 틀린 문제를 분석한다.
  3. 분석을 통해 오개념이나 몰랐던 개념들을 정리하고, 외운다.





























































통합사회

:

  1. 평소 수업에 집중하고 선생님이 말씀하신 내용을 필기한다.
  2. 교과서와 프린트, 필기를 완벽히 외운다.
  3. 심심할 때 문제도 풀어본다.





























































정보

:

  1. 평소에 배운 내용들을 열심히 복습해놓는다.
  2. 수업에 집중하고, 하루에 10분정도 코딩을 해본다.
  3. 선생님이 말해주신 내용들과 교과서, 프린트를 외운다.





























































한국사

:

  1. 수업시간에 수업에 열심히 참여한다.
  2. 선생님이 강조하신 내용들과 필기를 공책에 정리한다.
  3. 정리할 때, 공책 왼쪽에는 힌트를, 오른쪽에는 내용들을 쓴다.
  4. 왼쪽 힌트만 보고 오른쪽 내용을 다 쓸 수 있을 때까지 암기한다.
  5. 시험범위를 완벽히 암기한 후 문제도 풀어본다.
